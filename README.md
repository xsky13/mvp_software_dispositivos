# MVP sistema de manejo dispositivos IoT

Antes de correr instalar broker de mqtt mosquitto.

Servicios:

* Alertas: <https://github.com/xsky13/dispositivos_alert_manager>
* Dispositivos: [ https://github.com/lenny-amin/dispositivo_CRUD](https://github.com/lenny-amin/dispositivo_CRUD)
* Usuarios: <https://github.com/Abiel2310/dispositivos_usuarios.git>
* Api gateway: <https://github.com/xsky13/dispositivos_gateway>


Para correr el sistema usar start.sh (no olvidar cambiar variables de db)
